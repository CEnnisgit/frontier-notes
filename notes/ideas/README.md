# Ideas

Speculative ideas get their own file here: `NNN-short-name.md`.

Each idea file should include:

1. **Claim** — what the idea asserts, in one paragraph.
2. **Mechanism** — the proposed physical mechanism (equations preferred, hand-waves flagged).
3. **Predictions** — what it would predict that standard physics doesn't.
4. **Falsifiers** — experiments or observations that would kill it.
5. **Status** — have those experiments been run? what did they find?
6. **Nearest physics-allowed cousin** — if the idea is forbidden as stated, what's the closest real question?

Pressure-testing an idea is not the same as dismissing it. The goal is: if it survives, you know exactly why; if it dies, you know exactly how.
